## /login(登录接口)

### 接口描述

在数据库中根据关键词(用户名,邮箱,手机号)及密码查询相关用户，若查找成功则存入Session，表明登陆成功

### 接口属性

* 接口路径: /login
* 接口说明: 在数据库中根据关键词(用户名,邮箱,手机号)及密码查询相关用户，若查找成功则存入Session，表明登陆成功
* 返回格式: json
* 请求方式: post

### 请求示例

* 请求内容: keyword=test&passwd=test
* 响应内容: {code:0,msg:"OK"}

### 请求参数

| 名称    | 类型   | 是否必填 | 说明                       |
| ------- | ------ | -------- | -------------------------- |
| keyword | string | 是       | 关键字(用户名,邮箱,手机号) |
| passwd  | string | 是       | 密码                       |

### 返回参数

| 名称 | 类型   | 说明                         |
| ---- | ------ | ---------------------------- |
| code | int    | 响应代码                     |
| msg  | string | 成功返回OK，失败返回错误信息 |
