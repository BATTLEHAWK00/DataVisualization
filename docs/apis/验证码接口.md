## /vericode(获取验证码接口)

### 接口描述

获取验证码，并将验证码存入Session中，以便比对

### 接口属性

* 接口路径: /vericode
* 接口说明: 获取验证码
* 返回格式: img/png
* 请求方式: GET

## /vericode(比对验证码接口)

### 接口描述

将提交的验证码与Session中存储的验证码进行比对

### 接口属性

* 接口路径: /vericode
* 接口说明: 比对验证码
* 返回格式: json
* 请求方式: POST

### 请求参数

| 名称 | 类型   | 是否必填 | 说明   |
| ---- | ------ | -------- | ------ |
| code | string | 是       | 验证码 |

### 请求示例

* 响应内容: {code:0,msg:"OK"}

### 返回参数

| 名称 | 类型   | 说明                         |
| ---- | ------ | ---------------------------- |
| code | int    | 响应代码                     |
| msg  | string | 成功返回OK，失败返回错误信息 |
